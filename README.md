# Adopt a Pet – React Router Practice

This project is based on a Codecademy exercise to practice **client-side routing with React Router v6**. The app allows users to browse pets available for adoption and search for pets by name.

## What I practiced
- Creating routes with React Router
- Navigation using **Link**, **NavLink**, **Navigate**, and `navigate`
- Handling **query parameters** for search
- Understanding how **Mock Service Worker (MSW)** mocks API requests

## Extra improvements
- Clearing the search input after submitting a search
- Showing a message and image when no pets match the search query
- Adding navigation to the homepage when clicking the logo
